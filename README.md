# cython-examples

Example using Cypthon


## Build module

```
./build-module.sh
```


You may need to install cython, i.e., `pip install cython` or `pip3`

## Run module

Start python and run the module as follows

```
Python 2.7.12 (default, Jun 29 2016, 14:05:02) 
>>> import hello_caller
>>> hello_caller.hello()
Hello world!
>>> 
```
